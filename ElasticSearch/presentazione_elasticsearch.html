Cos'è Elasticsearch?

Elasticsearch è un database NoSQL documentale orientato alla ricerca. Non ci sono tabelle né join: ogni personaggio è un documento JSON autonomo. La sua forza è la ricerca full-text sul testo e le aggregazioni sui dati numerici.

📄

Documenti, non righe

Ogni personaggio = un documento JSON con i suoi campi. Nessuno schema rigido, nessun join.

🔎

Ricerca full-text

I campi text vengono analizzati (spezzati in termini): puoi cercare parole nel testo, con punteggio di rilevanza.

📊

Aggregazioni

Raggruppa e calcola al volo: es. la taglia media per ciurma, senza estrarre i dati.

    Riepilogo
    
Cosa abbiamo visto

    

      
🔎 Full-text vs esatto
I campi text si cercano con match/multi_match (analizzati, con rilevanza e boost); i campi keyword con term (valore esatto).

      
🎯 Filtri e range
term per le categorie (ciurma), range per i numeri (taglia): la base di ogni interrogazione mirata.

      
📊 Aggregazioni
Con terms + avg otteniamo statistiche (taglia media per ciurma) senza estrarre i documenti.

      
⚡ REST + JSON
Ogni query è una chiamata HTTP con corpo JSON: la stessa che questa pagina invia in tempo reale a localhost:9200.
